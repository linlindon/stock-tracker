# UI 規格文件

---

## 一、設計原則

- **簡潔直接**: 個人使用，不需要過度設計
- **響應式**: 手機和桌面都能良好操作
- **資訊優先**: 關鍵數據要一目了然
- **操作流暢**: 常用功能要容易觸及

## 八、名詞說明區（每頁共用元件）

每個主要頁面底部放置一個可收合的說明區，列出該頁出現的專有名詞，適合投資新手查閱。

### 設計規格

```
┌─────────────────────────────────┐
│  📖 名詞說明          [收合 ▲] │ ← 預設展開，可收合
├─────────────────────────────────┤
│                                 │
│  平均成本 Average Cost           │
│  持有期間所有買入成本的加權平均， │
│  公式：總買入成本 ÷ 持有股數     │
│                                 │
│  已實現損益 Realized P/L        │
│  賣出後實際獲得或虧損的金額，     │
│  公式：賣出收入 - 賣出成本       │
│                                 │
│  已實現報酬率 Realized Return   │
│  公式：已實現損益 ÷ 賣出成本     │
│                                 │
└─────────────────────────────────┘
```

### 各頁說明內容

| 頁面 | 說明的名詞 |
|------|-----------|
| Dashboard | 總投入成本、持股成本、已實現損益、已實現報酬率、持股占比 |
| Holdings | 平均成本、持股總成本、最近買入價、目標價 |
| Transactions | 總金額、手續費、淨金額 |
| Realized P/L | 平均成本、賣出收入、成本基準、已實現損益、已實現報酬率 |
| 短線配對 | 自選成本基準、自選損益、自選報酬率，以及與均成本報酬率的差異說明 |

### 說明區互動

- 預設展開
- 點擊標題列可收合，狀態存在 `localStorage`（下次開啟保持上次的展開/收合狀態）
- 手機版收合後不佔空間
